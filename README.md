# 나만의 퀴즈 게임

Python 기본 문법과 Git 사용 흐름을 연습하기 위한 콘솔 퀴즈 게임입니다.

## 프로젝트 개요

터미널에서 메뉴를 선택해 퀴즈를 풀고, 새 퀴즈를 추가하고, 저장된 퀴즈 목록과 최고 점수를 확인할 수 있는 프로그램입니다.
퀴즈와 최고 점수는 프로젝트 루트의 `state.json` 파일에 저장되어 프로그램을 종료한 뒤 다시 실행해도 유지됩니다.

## 퀴즈 주제와 선정 이유

주제는 Python 기초 문법입니다.
이번 미션의 핵심 목표가 변수, 자료형, 조건문, 함수, 클래스 같은 Python 기본 개념을 직접 설명할 수 있게 되는 것이므로, 게임을 구현하면서 동시에 복습할 수 있는 주제로 선택했습니다.

## 실행 방법

Python 3.10 이상이 설치된 환경에서 아래 명령어를 실행합니다.

```bash
python main.py
```

## 기능 목록

- 메뉴 선택
- 퀴즈 풀기
- 퀴즈 추가
- 퀴즈 목록 확인
- 최고 점수 확인
- `state.json` 저장 및 불러오기
- 잘못된 숫자 입력, 빈 입력, `Ctrl+C`, EOF 입력 예외 처리
- 손상된 데이터 파일 복구

## 데이터 파일 설명

`state.json`은 프로젝트 루트에 위치하며 퀴즈 목록과 최고 점수를 저장합니다.
파일은 UTF-8 인코딩으로 읽고 씁니다.

```json
{
    "quizzes": [
        {
            "question": "문제 내용",
            "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
            "answer": 1
        }
    ],
    "best_score": 100,
    "best_correct": 5,
    "best_total": 5
}
```

- `quizzes`: 저장된 퀴즈 목록
- `question`: 문제 문장
- `choices`: 4개의 선택지
- `answer`: 정답 번호, 1부터 4까지의 정수
- `best_score`: 최고 점수
- `best_correct`: 최고 점수를 기록했을 때 맞힌 문제 수
- `best_total`: 최고 점수를 기록했을 때 전체 문제 수
