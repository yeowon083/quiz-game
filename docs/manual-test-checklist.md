# 수동 테스트 체크리스트

## 1) 기본 실행
- `python3 main.py` 실행 시 메뉴가 표시된다.
- `5` 입력 시 저장 후 정상 종료된다.

## 2) 입력 예외 처리
- 메뉴에서 `abc` 입력 시 재입력 안내가 나온다.
- 메뉴에서 `9` 입력 시 재입력 안내가 나온다.
- 메뉴에서 빈 입력(Enter) 시 재입력 안내가 나온다.

## 3) 퀴즈 추가
- 메뉴 `2`에서 문제/선택지/정답 입력으로 새 퀴즈를 등록할 수 있다.
- 등록 후 메뉴 `3`에서 목록에 반영된다.

## 4) 퀴즈 플레이
- 메뉴 `1`에서 등록된 문제들이 순서대로 출제된다.
- 정답/오답 판정과 최종 점수 출력이 정상 동작한다.

## 5) 점수 확인
- 메뉴 `4`에서 최고 점수가 표시된다.
- 최고 점수가 갱신되면 `state.json`에 반영된다.

## 6) 영속성
- 프로그램 종료 후 재실행해도 추가한 퀴즈와 최고 점수가 유지된다.
- `state.json` 손상 시 기본 데이터 복구 메시지와 함께 실행된다.
